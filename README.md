## Run Locally

Clone the project

```bash
  git clone https://github.com/primeaddict/todo-reactjs-redux.git
```

Go to the project directory

```bash
  cd todo-reactjs-redux
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
